# 🎮 Minecraft 2D – Single Player

A browser-based 2D survival sandbox game inspired by Minecraft.

Built using pure HTML, CSS, and JavaScript — no game engine required.

---

## 🌟 Features

- 🧱 Procedurally generated world (biomes, caves, ores)
- 🌞 Dynamic day & night cycle
- 👤 Custom animated player
- 🧟 Hostile & passive mobs
- 🎒 Inventory system (36 slots + hotbar)
- 🔨 Block breaking & placing
- 🛠 Crafting system
- ❤️ Health & hunger mechanics
- 🔥 Lava & water interactions
- ✨ Particle effects and item drops

---

## 🚀 How To Run

### Option 1 – Open Directly

1. Download the project
2. Open `minecraft2d.html`
3. Double-click the file

The game will run in your browser.

No installation required.
